# 핵융합 소재 AI 제안서 인간 검토 협력자 답변 점검 및 최종 반영 기준

작성일: 2026년 5월 31일  
대상: 강윤정 인간 검토 협력자 Q1~Q8 답변  
결론: 핵심 방향은 채택하되, 공식 제안서에는 과학적 과장과 투자 과장으로 읽힐 수 있는 표현을 낮춰 반영한다.

---

## 0. 역할 정의

강윤정은 본 프로젝트에서 단순한 문장 교정자가 아니라, AI 자동화 조직 안에서 초안을 읽고 대중성·현실감·설득 장면을 점검하는 인간 검토 협력자다.

이 역할은 완성된 권위자가 아니라 협력자이자 검토자다. 인간 검토에는 경험 편차, 맥락 누락, 과장 표현 삽입 가능성이 남아 있으므로, AI는 강윤정의 피드백을 그대로 수용하지 않는다. 대신 장점은 채택하고, 과학적 불확실성·사업 리스크·검증 게이트가 약한 부분은 다시 보완한다.

---

## 1. 종합 평가

강윤정 인간 검토 협력자의 답변은 일반인 대상 전달력과 후원자 설득 장면 구성에서 강점이 있다. 특히 “금속 조합 검색엔진”, “실험 내비게이션”, “내일 아침 실험실에서 무엇부터 검증할지 정한다”는 표현은 프로젝트의 즉시성과 용도를 빠르게 이해시키는 데 효과적이다.

다만 투자 제안서 본문에 그대로 넣기에는 몇 가지 위험이 있다. “독점적 소재 배합비”, “1초 만에 수만 개”, “연구비가 투입되는 즉시 DFT/CALPHAD 자동 연동”은 실제 데모와 연구 단계 사이의 거리를 줄여 보이게 만들 수 있다. 최종본에서는 이 표현들을 보수적으로 조정해야 한다.

평가 점수: 82/100

| 항목 | 평가 | 판단 |
| --- | --- | --- |
| 30초 이해도 | 높음 | 비전문가 설명으로 적합 |
| 민간 후원 설득력 | 높음 | IP와 데이터 자산 관점이 좋음 |
| 과학적 정확성 | 보통 | 일부 표현은 과장 가능성 있음 |
| 데모 시연성 | 높음 | 라이브 화면 중심 구성이 좋음 |
| 연구비-산출물 연결 | 보통 이상 | Phase 구조는 좋지만 산출물 검증 기준 보강 필요 |
| 투자 리스크 설명 | 부족 | 실패 가능성, IP 귀속, 데이터 품질 리스크가 약함 |

## 2. 채택할 표현

다음 표현은 최종 문서와 발표자료에 반영할 가치가 있다.

1. 금속 조합 검색엔진
2. 실험 우선순위 내비게이션
3. 수많은 후보 중 먼저 계산하고 실험할 후보를 줄이는 시스템
4. 말보다 먼저 설득하는 라이브 데모
5. 성공 데이터뿐 아니라 실패 데이터까지 회수하는 active learning 구조
6. 민간 연구조직이 자체 후보 조성 데이터와 검증 리포트를 축적하는 구조

## 3. 조정해야 할 표현

| 원문 방향 | 문제 | 최종본 기준 |
| --- | --- | --- |
| 독점적 소재 배합비와 공정 데이터 자산 선점 | 아직 검증 전 후보 단계이므로 과장 가능 | 특허 검토 가능한 후보 조성군과 데이터 자산 축적 |
| 1초 만에 수만 개 후보 | 환경에 따라 성능 차이가 있음 | 수백~수천 개 후보를 즉시 스크리닝 |
| DFT/CALPHAD가 연구비 투입 즉시 자동 연동 | Phase 1~2 구축 과제임 | 연구비 확보 후 단계적으로 계산 검증 파이프라인을 연결 |
| 톱니바퀴처럼 맞물림 | 제안서 문체로는 과함 | 단계별 예산과 산출물이 연결됨 |
| 완벽한 신소재를 발명했다는 표현 배제 | 방향은 맞음 | “최종 소재 선언이 아니라 후보 우선순위화”로 반복 |

## 4. 보완한 핵심 메시지

### 4.1 일반인용

> 핵융합로 안쪽은 극한의 열과 방사선을 견뎌야 합니다. 이 도구는 수많은 금속 조합 중에서 먼저 계산하고 실험해볼 후보를 빠르게 줄여주는 검색엔진입니다.

### 4.2 민간 후원자용

> 이 프로젝트의 가치는 후보 하나를 맞히는 데 있지 않습니다. 민간 연구조직이 후보를 만들고, 계산하고, 실험하고, 실패 데이터까지 회수하는 반복 구조를 갖는 데 있습니다.

### 4.3 과학적 신뢰 확보용

> 현재 데모는 실제 DuctGPT 모델이나 AtomGPT 가중치를 포함하지 않습니다. 조성 기반 물리 descriptor와 휴리스틱 surrogate로 만든 1차 스크리닝 도구이며, 모든 후보는 DFT·CALPHAD·합성·기계적 시험·핵융합 환경 검증을 거쳐야 합니다.

## 6. 다음 인간 검토 지시

다음 라운드 인간 검토 협력자는 문장을 더 화려하게 만드는 것보다 다음 기준을 지켜야 한다.

1. “AI가 신소재를 발견했다”가 아니라 “AI가 후보를 줄인다”로 쓴다.
2. IP는 “즉시 독점”이 아니라 “후보 조성군과 검증 데이터 축적”으로 쓴다.
3. Phase 0~3 예산은 투자자 선택지를 낮추는 장치로 유지한다.
4. 데모 장면은 반드시 실시간 URL 중심으로 설명한다.
5. 과학적 한계는 숨기지 말고 앞쪽에 둔다.
